#### 状态机是什么？
状态机是一种数学模型，用来表示状态以及状态之间的转换和动作。

我们一般使用的是确定性有限状态机。（即一种状态对于一种输入只能有一种转换规则）。


#### 状态机的组成要素？

一个确定性有限状态机有基本的五个组成部分：
- 输入
- 状态集
- 初始状态
- 结果状态机
- 转换规则
